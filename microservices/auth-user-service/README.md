# 🏐 SprintBot - Auth User Service

Microservice d'authentification et de gestion des utilisateurs.

## 🚀 Démarrage

```bash
docker-compose up -d
```

## 🛑 Arrêt

```bash
docker-compose down
```

## 📊 API

- `POST /api/auth/login` - Connexion
- `GET /api/users/profile` - Profil utilisateur
- `GET /actuator/health` - État du service
